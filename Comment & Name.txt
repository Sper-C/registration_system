Comment:
1) Trước tên hàm
Cần viết theo cấu trúc:
Chức năng hàm - Đầu vào của hàm - Đầu ra của hàm.
Nếu tên hàm đã được đặt dễ hình dung ra chức năng thì không cần comment.
2) Trong thân hàm
Chỉ comment tại các dòng code cho là khó hiểu.
Hoặc comment để phân chia các giai đoạn làm việc trong 1 hàm.
------------------------------------------------------------------------------
Name: (hàm, biến, struct,...)
1)Để giảm thiểu comment thì đặt tên hàm phải cụ thể, gợi nhớ, có thể dài. 
Ví dụ hàm có chức năng chuyển từ string sang int thì có thể đặt là:
int FromString_toInt, String_toInt hoặc ConvertString_Int,....

2)Tên biến cũng tương tự, hạn chế đặt tên biến là các chữ cái a,b,c,d....
Ví dụ biến biểu thị năm học nên là year, thay vì y. 
Tên biến nên rõ ràng, cụ thể và đồng bộ để dễ hình dung chức năng của nó.

Tên biến & struct của Quân:
1) Menu.cpp:
option  	- thể hiện sự lựa chọn của người dùng khi chọn trong menu (thuộc các hàm Hiển thị Menu).
run 		- thể hiện việc vận hành của các hàm menu nhỏ hơn. Nếu run là true thì chạy các menu nhỏ, false thì không (thuộc Valid_Data).
limit  		- số giới hạn sự lựa chọn trong các menu.
2) Year.cpp:
path 		- nói chung là đường dẫn đến file nào đó.
year_path 	- đường dẫn đến file csv của năm.
year_name 	- tên năm, là tên của file csv (Vd: 2020-2021.csv).
store 		- một file csv lớn lưu giữ các file csv năm.
years_path 	- chính là store.
quanti 		- số lượng các đối tượng hiển thị (ví dụ số năm, số lớp học).
3) Class.cpp
class_path 	- đường dẫn đến file csv của lớp.
class_name 	- tên lớp, là tên file csv.
dir 		- đường dẫn tới một thư mục.
file_path(file) - đường dẫn tới một file bất kỳ.
check 		- một đường dẫn file cho trước dùng để kiểm tra sự trùng lặp.
depart 		- thể hiện sự lựa chọn khoa của trường.
system 		- hệ đào tạo.
4) Node Proccess.cpp
info 		- thông thường là thông tin của một node
node  		- con trỏ node
list 		- một danh sách liên kết
5) Ở MainSource.h
stuct user 	- struct để lưu dữ liệu người dùng, có thể mở rộng để chứa thêm nhiều loại thông tin.
struct paths	- danh sách liên kết chứa các node lưu đường dẫn.
